# Magic Converter

Harry Potter World currency converter.

## Rate:

-   1 Galleon = £4.93
-   1 Sickle = £0.29
-   1 Knut = £0.01

GBP to RUB exchange rate taken from the [API](https://www.cbr-xml-daily.ru/daily_json.js) of the Central Bank of Russia.

## Launch
```sh
flask --app magic_converter run 
```
